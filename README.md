# index-wukong-archive

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为静态 HTML 资源的集中存储与展示空间。  
所有页面均为独立文件，彼此之间无关联依赖，便于长期保存与快速访问。

## 目录说明

```
/
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放归档的 HTML 页面
│   ├── example-01.html
│   ├── example-02.html
│   └── ...
├── assets/             # 页面引用的静态资源（可选）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，文件名通常反映页面内容或归档时间。
- `assets/` 目录用于存放页面可能引用的外部样式、脚本或图片资源，非必需。
- 根目录下的 `index.html` 可作为入口页面，提供页面列表或简单导航。

## 页面归档说明

- 每个 HTML 文件均为独立完整页面，可脱离仓库环境直接打开浏览。
- 页面内容来源于历史存档、个人项目、学习笔记或其他公开资料，不涉及隐私或敏感信息。
- 归档以“保留原始状态”为原则，页面内容和结构通常不做修改，仅作必要清理（如移除失效链接）。
- 部分页面可能包含外部引用资源（如字体、图片），若资源失效，页面显示可能受影响，但不影响 HTML 本身的结构。
